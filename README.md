# Spotify_clone-HTML-CSS
A simple Spotify web page clone using HTML and CSS. no JavaScript or Backend functionality, just a single-page design.

## Features
- Replicates Spotify’s landing page layout
- Clean HTML and CSS structure
- Beginner-friendly project

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any web browser.
